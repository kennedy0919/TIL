# 파이썬 함수 정리
1. len() : 함수의 원소가 몇개 들어있는지 알려줌

2. 원소[숫자] : 원소안에 숫자번째 원소가 무엇인지 알려줌 (주의할점 :원소의 숫자는 0부터 시작함)

3. remove() : 리스트.remove(원소) 를 작성하면 리스트에서 원소가 삭제됨

4. while : 어떤 조건이 만족되는동안 그 아래 쓴 문장들을 반복함

5. if : 만약
6. elif : if 다음

7. for : 시퀀스를 이용해서 원하는 명령을 반복할 때 쓰입니다. 
#### 시퀀스가 근데 뭐지?

8. range(숫자1,숫자2) : 숫자 1과 숫자2 를 포함한 그안의 정수를 나열해서 리스트로 만듬
